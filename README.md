# 🎓 AI-платформа для подготовки к ОГЭ/ЕГЭ

Персонализированная платформа для подготовки к экзаменам с AI-репетитором в стиле Т-Банка.

## 🚀 Технологии

- **Frontend**: Next.js 14+, TypeScript, Tailwind CSS
- **Backend**: Firebase (Auth, Firestore, Functions, Storage)
- **AI**: OpenAI GPT-4, RAG архитектура
- **Design**: Т-Банк стиль (#FFE600, Inter font)

## 📁 Структура проекта

```
/prototypes     - HTML прототипы (test.html, results.html)
/описание       - Исходное техническое задание
TASKS.md        - Roadmap разработки
ARCHITECTURE.md - Архитектура системы
DESIGN_SYSTEM.md - Дизайн-система
AI_PROMPTS.md   - Промпты для AI
CONTENT_TEXTS.md - Тексты для сайта
USER_FLOW.md    - User flow диаграммы
```
