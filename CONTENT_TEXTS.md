## 3. FAQ

### Общие вопросы

**Q: Что такое AI-Наставник ЕГЭ?**
```
A: Это онлайн-платформа для подготовки к ОГЭ и ЕГЭ с использованием искусственного интеллекта. AI анализирует твои ошибки, создаёт персональный план и объясняет сложные темы как настоящий репетитор.
```

**Q: Сколько это стоит?**
```
A: Базовый функционал доступен бесплатно: 10 AI-объяснений в день, 20 сообщений в чате и 3 теста. Premium подписка (от 500₽/мес) даёт безлимитный доступ ко всем функциям.
```

**Q: Какие предметы доступны?**
```
A: Сейчас доступны все основные предметы ЕГЭ и ОГЭ: математика (база и профиль), русский язык, обществознание, физика, информатика, биология, химия, история, литература, география, английский язык.
```

**Q: Как работает AI-разбор ошибок?**
```
A: После каждого теста AI анализирует твои ответы, находит ошибки и объясняет:
- Где именно ты ошибся
- Почему это неправильно
- Как решать правильно (пошагово)
- Что повторить для закрепления

Всё это на простом русском языке, без сложных терминов.
```

**Q: Можно ли использовать на телефоне?**
```
A: Да! Платформа полностью адаптирована под мобильные устройства. Скоро выйдет и мобильное приложение для iOS и Android.
```

### Про подготовку

**Q: Сколько времени нужно заниматься?**
```
A: Рекомендуем 30-60 минут в день. AI составит реалистичный план с учётом твоего свободного времени.
```

**Q: За сколько до экзамена нужно начинать?**
```
A: Чем раньше, тем лучше! Оптимально начать за 6-9 месяцев. Но даже за 2-3 месяца можно значительно улучшить результат.
```

**Q: Заменит ли AI репетитора?**
```
A: AI — отличный помощник для самостоятельной подготовки. Он доступен 24/7, персонализирует обучение и намного дешевле репетитора. Но для сложных случаев (очень низкий уровень, нужна жёсткая дисциплина) живой репетитор может быть полезен.
```

**Q: Гарантируете ли вы высокий балл?**
```
A: Результат зависит от твоих усилий. AI даёт все инструменты для эффективной подготовки, но заниматься нужно регулярно. Наша статистика показывает, что ученики, занимающиеся по плану 5+ дней в неделю, повышают балл на 15-25 пунктов за 3 месяца.
```
