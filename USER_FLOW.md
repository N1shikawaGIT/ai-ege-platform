## 🎯 Детальные потоки

### 1. Регистрация нового пользователя

```
Landing → Кнопка "Начать" 
    ↓
Регистрация (auth.html - вкладка "Регистрация")
    ↓
Ввод данных:
    • Имя
    • Email
    • Пароль
    • [✓] Согласие с условиями
    ↓
ИЛИ "Регистрация через Google"
    ↓
Onboarding (4 шага)
    ↓
Диагностический тест
    ↓
Результаты
    ↓
Dashboard
```

### 2. Вход существующего пользователя

```
Landing → Кнопка "Вход"
    ↓
Вход (auth.html - вкладка "Вход")
    ↓
Ввод данных:
    • Email
    • Пароль
    ↓
ИЛИ "Войти через Google"
    ↓
Dashboard (сразу, без onboarding)
```

### 3. Прохождение теста

```
Dashboard → "Новый тест"
    ↓
Выбор типа:
    • Диагностический
    • Практика по темам
    • Пробный экзамен
    ↓
Страница теста (test.html)
    ↓
Решение заданий:
    • Ввод ответа
    • Навигация (1-20)
    • Можно вернуться
    ↓
Кнопка "Завершить"
    ↓
Результаты (results.html)
    ↓
Действия:
    • "Перейти в кабинет" → Dashboard
    • "Задать вопрос AI" → Chat
```

### 4. Работа с AI-чатом

```
Dashboard → "AI-Репетитор"
    ↓
Чат (chat.html)
    ↓
Варианты:
    • Задать вопрос вручную
    • Выбрать готовый вопрос
    ↓
AI отвечает
    ↓
Диалог продолжается
    ↓
Можно вернуться в Dashboard
```

### 5. Просмотр результатов прошлого теста

```
Dashboard → История тестов
    ↓
Клик на тест
    ↓
Результаты (results.html)
    ↓
Просмотр разбора ошибок
    ↓
Клик "Показать AI-разбор" на конкретном задании
    ↓
Раскрывается объяснение
```

### 6. Работа с профилем

```
Dashboard → Профиль (боковое меню)
    ↓
Профиль (profile.html)
    ↓
Секции:
    • Личная информация
    • Статистика
    • Достижения
    • Подписка
    • Настройки
    ↓
Изменения → "Сохранить"
    ↓
Назад в Dashboard
```

## 🔄 Циклы и повторяющиеся действия

### Ежедневный цикл пользователя

```
1. Вход → Dashboard
2. Видит "План на сегодня"
3. Решает задания по плану
4. Получает AI-разбор ошибок
5. Задаёт вопросы в чате
6. Смотрит прогресс
7. Выход (возврат завтра)
```

### Подготовка к экзамену (долгосрочный цикл)

```
1. Регистрация + Диагностика
2. Получение персонального плана
3. Ежедневная практика (цикл выше)
4. Еженедельные проверочные тесты
5. Корректировка плана
6. Повторение до экзамена
```

## 🚪 Точки выхода

Пользователь может покинуть приложение из любого экрана:
- **Landing** → закрыл вкладку (не зарегистрировался)
- **Auth** → не завершил регистрацию
- **Onboarding** → не завершил настройку (прогресс сохраняется)
- **Test** → бросил тест (можно продолжить позже)
- **Dashboard** → выход (данные сохранены)

---

## ⚡ Быстрые переходы

Из любого места можно:
- **Dashboard** → главная страница (всегда доступна через меню)
- **Chat** → AI-помощь (из результатов, из теста)
- **Profile** → настройки (боковое меню)

---

## 📱 Мобильный flow (отличия)

На мобильных:
- Боковое меню → бургер-меню
- Навигация теста → вертикальная прокрутка
- Результаты → одна колонка (sidebar внизу)

Логика остаётся та же.

---

## 🎨 Визуальные переходы

### Анимации между экранами:
- **Landing → Auth**: fade + slide
- **Auth → Onboarding**: прогресс-бар появляется
- **Onboarding шаги**: slide влево при "Далее"
- **Test → Results**: fade out теста, fade in результатов
- **Dashboard**: все элементы fade in постепенно

## 🎯 Критические пути (Happy Path)

### Минимальный путь к ценности:
```
Регистрация (2 мин)
    ↓
Onboarding (1 мин)
    ↓
Диагностика (15 мин)
    ↓
AI-разбор ошибок (5 мин)
    ↓
Понимание слабых мест ✅

Итого: 23 минуты до первой ценности
```

### Ежедневное использование:
```
Вход (10 сек)
    ↓
Dashboard (5 сек)
    ↓
Начать практику (5 сек)
    ↓
Решение 5 заданий (10 мин)
    ↓
AI-разбор (3 мин)

Итого: 13 минут в день
```
